# 案头雅记 (Quantum Life Dashboard)

> 这是一个将中国古典美学、人工智能学习需求与自动化排程相融合的**个人数字基建 (Personal OS)**。

> 从浏览器走向桌面，以原生架构重塑极客与文人的数字案头。

## 🌟 设计哲学与架构演进 (v1.1.0 墨梅)

本系统摒弃了传统的“流水账式”待办列表，采用“ **全局仪表盘 (Dashboard)** ”思想。在最新 v1.1.0 版本中，系统实现了从「时间刻度管理」到「个人知识库构建」的跃迁，全面采用 **Electron + Vue 3 + Node.js** 的端到端桌面级双相架构：

* **双相视图跃迁与零延迟心流** ：废弃传统 `<component :is>` 动态路由可能带来的组件卸载白屏问题，全局采用 `v-show` 状态保活机制。工作时处于高效的“案头仪表盘 (Dashboard)”，灵感闪现时一键无缝切入无边框的“灵感书斋 (Zen Writing)”，实现真正的零延迟心流体验。
* **极致降噪与并排双入口设计** ：将低频工具折叠收纳。左侧保留清新淡雅的“灵感抽屉”用于翻阅历史卡片，右侧将“知识图谱”、“量子词库”与“开启书斋”快捷入口优雅并排，保证案头的清爽与专注。
* **数据主权与 AI 预研** ：彻底摆脱浏览器缓存限制，任务流与知识库严格解耦。数据以 `.json` 格式永久落盘于本地物理路径，底层 Schema 已预留 `ai_metadata` 字段，为后续接入本地大模型与向量数据库打下基础。

## ✨ 核心功能特性

### 1. 灵感书斋与多维知识库 (Zen Writing Center)

系统内置独立于任务流的个人内容库，并引入了“温故”与“知新”的双轨并行逻辑：

* **雅记卡片弹窗 (Notes Modal)** ：点击仪表盘左侧淡雅的“灵感书斋”抽屉，即刻呼出居中悬浮卡片。支持自动过滤 Markdown 标记，提供极净的「🔖 摘录」、「📔 日记」、「🍃 散文」、「💻 量子随笔」等多维体裁历史翻阅。
* **快捷执笔与心流衔接** ：在知识图谱模块旁设有快捷“✍️ 书斋”入口；或在仪表盘遇到心仪的诗句时点击 `[写随笔]`。系统均会平滑切入写作视图（诗词模式下会自动将诗句作为 Quote 引用块生成草稿），实现多入口到灵感沉浸的完美收拢。

### 2. 三相历法中枢 (Calendar Board)

系统内置基于**策略模式**的日历主板，支持三种维度的数据可视化无缝切换：

* **点阵模式 (Dot Mode)** ：每日微观视角。根据当天任务总数渲染排布圆点，直观展现吞吐量。
* **进度模式 (Progress Mode)** ：宏观甘特图视角。任务化身彩色进度条， **任务标题极简内嵌于色带之中** ，项目跨度与分类一目了然。
* **课表模式 (Week Timetable)** ：近期规划视角。提取未来7天的日程，采用紧凑堆叠的课表UI。

### 3. 智能任务引擎与色彩联动 (Task Engine & Color Sync)

任务系统深度解决“习惯打卡”与“单日代办”的逻辑冲突：

* **自动化流转** ：
* `Single` (单日任务)：未完成触发“自动延期 (Rollover)”。
* `Daily` (每日习惯)：单日独立打勾验证，状态数据独立挂载，不发生周期污染。
* `Multi` (多日项目)：设定首尾期限，支持向内嵌套 Sub-tasks。
* **全端动态色彩演算** ： 录入任务时支持 5 种主题色（支持自定义 Hex）。底层采用透明度叠加算法（Hex + 1A / 33），自动生成淡色背景及边缘阴影，确保 100% 全端视觉色彩同步并契合留白美学。

### 4. 硬件与 AI 接口预留 (Future Proof)

通过 Electron 的底层能力与全新的数据 Schema，系统随时准备扩展：

* 支持串口通信（对接桌面温湿度计/电子墨水屏）。
* 预留本地 Python 脚本与 LLM 调用能力，计划实现 AI 情绪打标、智能摘要与向量知识库检索。

## 🚀 启动与构建指引

系统现已由 Web 应用升级为原生桌面软件，请按以下步骤操作：

### 1. 依赖安装

```bash
# 进入前端目录安装 Vue 相关依赖
cd frontend
npm install
```

### 2. 开发模式 (Development)

开发时建议同时启动前端热更新和本地数据服务：

```bash
# 终端 1：启动前端 UI (进入 frontend 目录)
cd frontend
npm run dev

# 终端 2：启动 Electron 桌面壳包载前端页面
cd frontend
npm run electron:dev
```

### 3. 一键打包输出 EXE (Production)

当你完成功能修改，需要将其打包为独立的 Windows 桌面软件时，只需在前端目录执行：

```bash
npm run build:exe
```

*构建完成后，前往 `release/win-unpacked/` 目录，双击 `案头雅记.exe` 即可享受你的专属数字工作台。*
