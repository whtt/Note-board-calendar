# 案头雅记 (Quantum Life Dashboard)

> 这是一个将中国古典美学、人工智能学习需求与自动化排程相融合的**个人数字基建 (Personal OS)**。
>
> 从浏览器走向桌面，以原生架构重塑极客与文人的数字案头。

## 🌟 设计哲学与架构演进

本系统摒弃了传统的“流水账式”待办列表，采用“ **全局仪表盘 (Dashboard)** ”思想。在最新版本中，系统已全面重构为 **Electron + Vue 3 + Node.js** 的端到端桌面级架构：

* **解耦视图** ：将数据计算引擎与 UI 层剥离，前端、后端、桌面壳独立维护。
* **降噪设计** ：将低频工具折叠收纳；进度视图采用无列表的全宽甘特图，保证案头的清爽与专注。
* **数据主权** ：彻底摆脱浏览器缓存限制，数据以 `.json` 格式永久落盘于本地物理路径，为后续接入本地 AI 大模型与向量数据库打下基础。

## ✨ 核心功能特性

### 1. 三相历法中枢 (Calendar Board)

系统内置基于**策略模式**的日历主板，支持三种维度的数据可视化无缝切换：

* **点阵模式 (Dot Mode)** ：每日微观视角。根据当天任务总数渲染排布圆点，完成点亮，直观展现吞吐量。
* **进度模式 (Progress Mode)** ：宏观甘特图视角。去除了冗余的侧边文字，让 7 天时间轴横跨全宽。任务化身彩色进度条， **任务标题极简内嵌于色带之中** ，项目跨度与分类一目了然。
* **课表模式 (Week Timetable)** ：近期规划视角。提取未来7天的日程，采用紧凑堆叠的课表UI。

### 2. 智能任务引擎与色彩联动 (Task Engine & Color Sync)

任务系统支持三种基本形态与全量色彩映射：

* **自动化流转** ：

  * `Single` (单日任务)：未完成则触发 **“自动延期 (Rollover)”** 机制，跟随至当前的真实日期。
  * `Daily` (每日习惯)：支持 **单日独立打勾验证** （状态数据独立挂载，不发生周期污染）。
  * `Multi` (多日项目)：设定首尾期限。支持向内嵌套 Sub-tasks (子任务) 并双向联动。
* **动态色彩演算引擎 (Dynamic Color Mapping)** ：

  * 右侧案头雅记新增录入时支持**5种主题色**选取。任务卡片的左侧粗边框、打勾状态按钮，以及左侧进度视图的甘特图色带，将进行 **100%全端视觉色彩同步** 。
  * 引入极客风 HTML5 调色板，支持无限级别的自定义 Hex 颜色录入。底层采用 CSS 伪类与透明度叠加算法（Hex + 1A / 33），自动生成主色、淡色背景及边缘阴影，确保所有用户自定义色彩均完美契合案头的留白美学。

### 3. 应用百宝箱与灵感笔记 (Module Drawer & Notes)

右侧控制台采用 **折叠式模块中心** ，保持界面的克制与留白：

* **案头随记 (Memo)** 与 **诗词卡片 (Poetry)** 默认常驻。
* **🔖 灵感笔记卡片库** ：在诗词模块点击书签，即可将佳句摘录并生成精美的排版卡片，永久收录在百宝箱的笔记库中。
* **硬件与 AI 预留接口** ：通过 Electron 的底层能力，未来可轻松扩展串口通信（对接桌面温湿度计/墨水屏）或调用本地 Python 脚本分析数据。

## 🛠️ 技术栈

* **Desktop Shell** ：Electron
* **Frontend** ：Vue 3 (Composition API) + Vite + Tailwind CSS
* **Backend** ：Node.js + Express
* **Data Storage** ：本地 JSON 物理直写（纯正的数据主权，支持任意网盘挂载同步）

## 🚀 启动与构建指引

系统现已由 Web 应用升级为原生桌面软件，请按以下步骤操作：

### 1. 依赖安装

```bash
# 在项目根目录安装 Electron 等依赖
npm install
# 进入前端目录安装 Vue 相关依赖
cd frontend
npm install
```

### 2. 开发模式 (Development)

开发时建议同时启动前端热更新和本地数据服务：

```bash
# 终端 1：启动本地数据引擎
cd backend
node server.js
# 终端 2：启动前端 UI (进入 frontend 目录)
cd frontend
npm run dev
# 终端 3：(可选) 启动 Electron 桌面壳包载前端页面
npm run electron:dev
```

### 3. 一键打包输出 EXE (Production)

当你完成功能修改，需要将其打包为独立的 Windows 桌面软件时，只需在**根目录**执行

```bash
npm run build:exe
```

*构建完成后，前往 `release/win-unpacked/` 目录，双击 `案头雅记.exe` 即可享受你的专属数字工作台。*
